# Contributing

memloom is early. The most useful contribution right now is a file-format
extractor. See the extractors guide in the docs.

Two rules:

1. Sign the [CLA](./CLA.md) once. A bot asks on your first PR, you reply, done.
2. AI-assisted PRs are fine. This project is built with AI. But you have to be
   able to explain every line you submit. If you can't, the PR gets closed.

Dev setup is in the README. Tests run on real Postgres through PGLite, no Docker.
